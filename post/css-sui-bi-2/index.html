CSS过渡

通过transition设置过渡时间。利用选择器造成样式变化，形成渐变效果。

.login_register {
  transition: height 1s;
  -moz-transition: height 1s;
  /* Firefox 4 */
  -webkit-transition: height 1s;
  /* Safari and Chrome */
  -o-transition: height 1s;
  /* Opera */
}
