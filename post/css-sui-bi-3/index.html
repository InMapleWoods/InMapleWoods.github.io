居中效果

利用transform来实现水平垂直居中效果
transform: translate(-50%, -50%);
元素以自身的左上角为原点，往上往左分别移动自身长宽的50%，以使其居于中心位置。
与负的margin-left和margin-top实现居中不同，margin-left和margin-top必须知道自身的宽高，而translate可以在不知道宽高的情况下进行居中。
